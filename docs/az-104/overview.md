## Exam Overview

- **Exam:** AZ-104 — *Microsoft Azure Administrator*
- **Certification:** Microsoft Certified: Azure Administrator Associate
- **Level:** Associate (Intermediate)
- **Role:** Administrator
- **Passing score:** 700 / 1000
- **Renewal:** annual (free)
- **Cost:** price varies by country/region (~$165 USD in the US)

## Why This Exam Matters for AI Architects

AZ-104 is the infrastructure foundation and the **prerequisite for AZ-305 Expert**. AI workloads need network isolation (VNet, NSG, private endpoints), storage (ADLS Gen2, Key Vault), and identity/compute (managed identities, RBAC) — all AZ-104 topics. You cannot design secure AI solutions without understanding the infra layer.

## Skills Measured (verified areas)

Per the official study guide, AZ-104 measures these areas. Weights change between versions — **[the official study guide](https://learn.microsoft.com/en-us/credentials/certifications/exams/az-104/) is the source of truth**:

- Manage **Azure identities and governance**
- Implement and manage **storage**
- Deploy and manage **Azure compute** resources
- Implement and manage **virtual networking**
- **Monitor and maintain** Azure resources

## How to Study (learning-science approach)

1. **Diagnose first** with the official practice assessment.
2. **Do the labs.** AZ-104 is hands-on: practice in the Azure portal, CLI, PowerShell, and Bicep. Use a free Azure account.
3. **Active recall + spacing.** Reconstruct each admin workflow from memory; revisit across days (Dunlosky et al., 2013).
4. **Teach it back** with a short runbook per domain.
5. **Re-test to 80%+** before booking.
