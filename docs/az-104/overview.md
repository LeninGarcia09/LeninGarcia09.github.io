# AZ-104: Azure Administrator Associate

> **⚠️ This track is under construction.** Challenges coming soon.

## Exam Overview

- **Exam:** AZ-104
- **Level:** Associate
- **Target Audience:** Azure Administrators, DevOps Engineers
- **Cost:** $165 USD

## Why This Exam Matters for AI Architects

AZ-104 is the infrastructure foundation. AI agents need network isolation (VNet, NSG, private endpoints), storage (ADLS Gen2, Key Vault), and compute (managed identities, RBAC) — all AZ-104 topics. You cannot design secure AI solutions without understanding the infra layer.
