# 🗓️ Plan de Estudio de 12 Semanas

> 🌐 **Idioma:** Español · [English version](../study-plan)
>
> Un **apoyo de estudio** (~1 hora/día · 84 horas). No es una especificación oficial del examen — es una ruta sugerida. Combina los [cursos oficiales](./courses) con los [dominios](./domains).

## Fase 1: Fundamentos (Semanas 1–4)

| Semana | Enfoque | Practice Test |
|--------|---------|--------------|
| S1 | Agentic Loops y API base | Test 1 — Agentic Loops (10p) |
| S2 | Orquestación Multi-Agente | Test 2 — Multi-Agent Systems (10p) |
| S3 | Hooks, Workflows y Sesiones | Test 3 — Hooks & Sessions (10p) |
| S4 | Tool Design y MCP | Test 4 — Tool Design & MCP (10p) |

## Fase 2: Conocimiento Aplicado (Semanas 5–8)

| Semana | Enfoque | Practice Test |
|--------|---------|--------------|
| S5 | Configuración de Claude Code | Test 5 — Claude Code Config (10p) |
| S6 | Plan Mode, Iteración y CI/CD | Test 6 — Plan Mode & CI/CD (10p) |
| S7 | Prompt Engineering y Output Estructurado | Test 7 — Prompt Engineering (10p) |
| S8 | Validación, Batch y Multi-Pass | Test 8 — Validation & Multi-Pass (10p) |

## Fase 3: Preparación del Examen (Semanas 9–12)

| Semana | Enfoque | Practice Test |
|--------|---------|--------------|
| S9 | Gestión de Contexto | Test 9 — Context & Reliability (10p) |
| S10 | Contexto Avanzado y Procedencia | Test 10 — Advanced Context (10p) |
| S11 | Integración y Ejercicios Hands-On | Practice Exam 1 — Full (50p) |
| S12 | Preparación Final | Practice Exams 2 & 3 — Full (50p c/u) |

**Ejercicios Hands-On de la Semana 11:**
1. Agente Multi-Tool con lógica de escalamiento
2. Configuración de flujo de equipo con Claude Code
3. Pipeline de extracción de datos estructurados
4. Pipeline de investigación multi-agente
