# SC-500: Microsoft Cybersecurity Architect

> **⚠️ This track is under construction.** Challenges coming soon.

## Exam Overview

- **Exam:** SC-500 (replaces SC-100 in 2025)
- **Level:** Expert
- **Target Audience:** Security Architects, Cloud Security Engineers
- **Cost:** $165 USD

## Why This Exam Matters for AI Architects

SC-500 covers **Microsoft Defender for AI**, **Microsoft Purview** (data governance + AI Hub), and **Zero Trust architecture** — all critical when building enterprise AI solutions. As an AI architect, you need to know how to secure the AI stack from identity to network to data.

## Skills Measured

| Domain | Weight |
|--------|--------|
| Identity and access management | 20-25% |
| Security for storage, databases, and data flows | 25-30% |
