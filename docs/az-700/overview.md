# AZ-700: Azure Network Engineer Associate

> **⚠️ This track is under construction.** Challenges coming soon.

## Exam Overview

- **Exam:** AZ-700
- **Level:** Associate
- **Target Audience:** Network Engineers, Infrastructure Architects
- **Cost:** $165 USD

## Why This Exam Matters for AI Architects

Private endpoints, VNet peering, ExpressRoute, DNS private zones — these are the networking primitives that enable enterprise AI isolation. Challenge 01 in the AI Foundry track uses private endpoints and private DNS zones. AZ-700 gives you the depth to design and troubleshoot these confidently.

## Skills Measured

| Domain | Weight |
|--------|--------|
| Core networking (VNet, NSG, routing) | 25-30% |
| Connectivity services (VPN, ExpressRoute) | 20-25% |
| Application delivery (Load Balancer, App Gateway) | 15-20% |
| Private access (Private Link, Private Endpoint) | 10-15% |
| Network security (Firewall, DDoS, WAF) | 15-20% |
