## Exam Overview

- **Exam:** AZ-700 — *Designing and Implementing Microsoft Azure Networking Solutions*
- **Certification:** Microsoft Certified: Azure Network Engineer Associate
- **Level:** Associate (Intermediate)
- **Role:** Network Engineer
- **Passing score:** 700 / 1000
- **Renewal:** annual (free)
- **Cost:** price varies by country/region (~$165 USD in the US)

## Why This Exam Matters for AI Architects

Private endpoints, VNet peering, ExpressRoute, and DNS private zones are the networking primitives that enable enterprise AI isolation. Challenge 01 in the AI Foundry track uses private endpoints and private DNS zones. AZ-700 gives you the depth to design and troubleshoot these confidently.

## Skills Measured (verified areas)

Per the official credential, AZ-700 covers these areas. Weights change between versions — **[the official study guide](https://learn.microsoft.com/en-us/credentials/certifications/exams/az-700/) is the source of truth**:

- Design and implement **core networking infrastructure**
- Design and implement **hybrid connectivity** (VPN, ExpressRoute)
- Design and implement **application delivery** services (Load Balancer, App Gateway)
- Design and implement **private access** to Azure services (Private Link / Private Endpoint)
- Design and implement **network security** (Firewall, DDoS, WAF)

## How to Study (learning-science approach)

1. **Diagnose first** with the official practice assessment.
2. **Build the topologies.** Deploy hub-and-spoke, private endpoints, and DNS private zones in a test subscription.
3. **Active recall + spacing.** Reconstruct each networking flow from memory; revisit across days (Dunlosky et al., 2013).
4. **Diagram it.** Draw the packet path for each private-access scenario.
5. **Re-test to 80%+** before booking.
